# PLAKTAGO

Plaktago is a bingo mobile app develop in flutter

## Usage

1. Clone the repository   
   ```git clone git@github.com:Extauren/Plaktago-App.git```  

2. Install Flutter dependencies  
   ```flutter pub get```  

3. Run Plaktago app   
   ```flutter run```
